## 0. Tailwind ↔ px 변환표

| Tailwind class | px | 사용 |
|---|---|---|
| `text-[10px]` | 10 | 작은 badge, 표 sub |
| `text-[11px]` | 11 | 푸터 라벨, 표 sub |
| `text-xs` | 12 | 보조 텍스트, sub-text, badge |
| `text-[15px]` | 15 | 본문 (타입 설명) |
| `text-sm` | 14 | 라벨, 카드 항목명 |
| `text-base` | 16 | 본문, 섹션 헤딩, nav 버튼 |
| `text-lg` | 18 | 카드 합계, 모달 제목, 옵션 라벨 |
| `text-xl` | 20 | 로딩 페이지 헤딩 |
| `text-[22px]` | 22 | 푸터 총액 |
| `text-2xl` | 24 | 카드 큰 금액, 타입 진단 헤딩 |
| `text-[28px]` | 28 | 시뮬레이션 질문 타이틀 (mobile) |
| `text-[30px]` | 30 | 시뮬레이션 질문 타이틀 (sm+) |

### Weight

| Tailwind | weight | 의미 |
|---|---|---|
| `font-normal` | 400 | 보조 텍스트 |
| `font-medium` | 500 | 라벨, 카드 항목 |
| `font-semibold` | 600 | 헤딩, 강조 |
| `font-bold` | 700 | 큰 헤딩, 금액 |

### 디자이너 검토 권장 포인트

| 항목 | 현황 | 가능한 개선 |
|---|---|---|
| 12px·14px 빈번 사용 | 같은 사이즈인데 weight·색이 다양해 hierarchy 약함 | 12px·14px 사용처 정리, 명확한 역할 부여 |
| `text-[15px]` 유일 사용 | 타입 설명만 15px — scale에 없는 값 | 14px 또는 16px로 통일 검토 |
| 22px vs 24px | 푸터 금액 22 / 카드 금액 24 — 차이가 미세 | 둘 중 하나로 통일? |
| `text-[30px]` | 시뮬레이션 질문 타이틀 — 모바일에선 큼 | 28px로 줄이면 wrap 빈도 ↓ |
| 색상 단계 | `#171717`·`#373737`·`#525252`·`#737373`·`#A1A1A1` 5단계 grey | text-token 5단계 정착 |
