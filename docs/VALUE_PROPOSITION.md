# Value Proposition

> **Status**: Baseline (도출 완료)
> **Source**: Value Proposition Workshop (Before / After / Position / Statement, 4 step)
> **Last updated**: 2026-04-30

## Statement (외우는 1~2 문장)

> **결혼 막 결정한 사람이 *3분 안에 첫 예산 그림*을 손에 쥐도록, *웨딩 플래너 컨설팅 중 데이터·기능 영역만* 무료로 떼어낸 도구. 사람의 역할은 그대로 둠.**

## 5 Layer Breakdown

### Target
결혼 막 결정한 사람 (D persona)

### Before State
> "결혼이 닥쳤는데 *어디서 시작*도 *남들 평균*도 모르는 *정보 결핍 스트레스*"

세부 정서 구성:
- 정보 결핍: 어디서·뭘·언제·얼마·어떻게
- 절차 결핍: 순서·타이밍
- 사회적 비교 갈증 (가장 강한 emotion source)
- 사회적 학습 욕구 (꿀팁·조심해야 할 것)
- 결과 정서: **스트레스**

### After State
> "막막함이 *그림*이 되고, 그림이 *내 기준*이 되어, 결혼을 *시작할 수 있다*는 **확신**"

Transformation arc (5 단계 ladder):
1. 레퍼런스 확보
2. 인지·파악
3. 통제 가능으로 전환
4. 자기 기준 + 자기 이해 형성
5. **확신**

종착 정체성 전환: *결혼을 모르는 사람 → 내 결혼을 알아가는 사람*

### Position vs Alternatives
> "웨딩 플래너가 사람으로 해주던 일 중 *데이터·기능으로 자동화 가능한 부분만* 떼어내, *모두에게 무료로* 제공하는 첫 reference point."

| 대체 도구 | 갭 |
|---|---|
| Google Sheet (Excel) | 항목·시세 모름, 시간 부담 |
| 친구·선배에게 물어보기 | 1명 sample, 본인 조건과 다를 수 있음 |
| 웨딩 박람회 | 업체 광고 중심, 전체 그림 X |
| 웨딩 플래너·컨설팅 | 비용 비쌈, 정보 비대칭 |
| SNS 후기 (인스타·블로그) | 신뢰성 낮음, 케이스 비교 어려움 |
| 카페·커뮤니티 | 단편·익명, 정리 안 됨 |

**3축 동시 차별화**: 비용 0 + 즉시 + 통계 기반

### Philosophy (self-imposed boundary)

> **사람 역할 대체 X. 데이터·기능으로 제공할 수 있는 편의만 제공.**

이 boundary의 함의:
- 향후 AI 컨설팅 챗봇 추가 X
- 향후 플래너 매칭 서비스 추가 X
- 컨설팅·관계·맞춤 상담은 *우리 scope 외*

## 의사결정 가이드 (이 baseline 활용법)

- **Cycle 2 implementation 결정**: 변경이 transformation arc 어느 layer를 강화하는지 검증
- **카피·UX writing**: "막막함 → 확신" 정서 호와 일관되게
- **Roadmap scope 결정**: Philosophy boundary로 *안 할 것* 자동 필터
- **합류자 onboarding**: Statement 외우기가 1순위. 5 layer는 부속 학습
