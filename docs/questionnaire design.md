    # 결혼 준비 온보딩 질문 + 예산 설계 마스터 문서 v6.0 (최종)
    
    온보딩 13문항 → 유형 결과 → 추가금 케어 토글(유형별 자동 디폴트)
    
    ---
    
    ## 전체 흐름
    
    ```
    STEP 1. 온보딩 13문항      → 유형 + Modifier 측정
    STEP 2. 유형 결과 + 종합설계서  → 자동 분류, 기본 예산 범위
    STEP 3. 추가금 케어 (토글)     → 유형별 디폴트 ON/OFF 자동 세팅, 사용자 수정
    ```
    
    > 2단계 질문(R)은 별도로 묻지 않습니다. 유형 결과가 추가금 케어 페이지의 토글 기본값을 자동으로 세팅하고, 사용자가 끄고 켜며 실시간으로 예산을 조정합니다.

    # 【예산 계산 공식】
    
    ## 1. 예식장 식대 (보증인원 방식)
    
    ```
    보증인원 = 기본식대(총금액) ÷ 1인당 식대(대인)   [DB 자동 도출]
    
    IF 하객 수(M2) ≥ 보증인원:
        식대 = 하객 수 × 1인당 식대 ,  대관비용 = 면제
    IF 하객 수(M2) < 보증인원:
        식대 = 기본식대(총금액) ,  대관비용 = 별도 부과
    ```
    
    ## 2. 예식장 장식
    
    ```
    기본 장식비 = 항상 자동 포함 (필수, 기본 수준)
    생화 꽃장식 ON 시 = 기본 장식비 + 생화 비용 전액 가산  [차액 아님]
    ```
    
    ## 3. 전체 예산
    
    ```
    총 예상 예산 =
      [예식장]
        + 식대 (보증인원 공식)
        + 기본 장식비 (필수) [+ 생화 꽃장식 전액, 토글 ON 시]
        + 대관비용 (보증인원 미달 시)
        + 토글 ON 항목: 부케·플라워샤워·포토테이블·웨딩케이크
        + 토글 ON 항목: 사회자·주례·축하공연·도우미·폐백음식·폐백수모·한복
        + 본식 촬영 (베이스 선택값: 예식장 연계 30~80만 / 외부 전문 70~150만+)
      [스드메] × 지역계수(M5)
        + (스튜디오 136만 + 토글 ON 옵션)
        + (드레스 베이스 + 드레스 지정 + 토글 ON 옵션)
        + (메이크업 베이스 등급 + 혼주·헤어변형 토글)
      [기타 — 별도 입력]
        + 예물/예단 + 혼수 + 신혼여행
    ```
    
    > 추가금 케어 페이지: 디폴트 합계 표시 → 사용자가 토글 OFF 시 실시간 차감 → 하단 '예상 총 예산' 즉시 갱신.

    # 【데이터 처리 결정 사항】 (전체 확정)
    
    | 항목 | 결정 |
    | --- | --- |
    | 식대 계산 | (나) 보증인원 max 방식, 미달 시 대관 부과 |
    | 보증인원 | 기본식대(총금액) ÷ 1인당 식대로 자동 도출 |
    | 기본 장식비 vs 생화 | 기본 장식비 필수 포함 + 생화는 전액 추가(차액 아님) |
    | 혼주 메이크업 | 메이크업샵 기준(R5). 예식장 혼주 항목 제외 (더블카운팅 방지) |
    | 본식 촬영 | 유형별 디폴트 (경험연출=외부 전문 / 그 외=예식장 연계), 사용자 변경 가능 |
    | 본식 도우미 = 본식드레스도우미 | 동일 역할로 통합 |
    | 축가/축하공연 + 축주비 | '축하공연 섭외' 하나로 통합 |
    | 본식 워판 | '본식 원본' 오타 수정 |
    | R 질문 처리 | 별도 질문 제거 → 유형별 토글 디폴트로 전환 |
