# Experience Roadmap

## 프로덕트 비전
결혼 준비 중인 사용자가 결혼 유형과 조건을 선택하면, 통계 기반 평균값으로 예산 초안을 자동 생성해주는 웹앱.
"구상 상태"에서 "초안 작성 시작 상태"로의 전이를 돕는다.
시뮬레이션 결과의 숫자 간극을 실사례 콘텐츠로 메워주어 외부 이탈을 방지하고 저장/공유 전환을 유도한다.
