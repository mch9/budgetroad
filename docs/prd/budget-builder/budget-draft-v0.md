# Budget Draft (예산 초안 생성)

## Step 0. Feature Hierarchy
budget-builder

## Step 1. User Goal
내 결혼에 전체적으로 얼마가 드는지, 항목별로 감을 잡는 것

## Step 2. How This Helps
결혼 유형을 선택하면 통계 기반 평균값으로 항목별 예산을 자동 구성해줌으로써,
사용자가 직접 시세를 조사할 필요 없이 즉시 전체 예산 감을 잡을 수 있게 한다.

## Step 3. Emotional Journey
호기심 → 자기 투영 → 발견의 쾌감 → 안도/자신감
- "막막함"이 "감이 잡힘"으로 전환되는 순간이 핵심

## Step 4. Internal Feature Name
budget-draft

## Step 5. Functional Requirements

### 입력 단계
1. 사용자는 결혼 유형 항목들을 선택할 수 있어야 한다
   - 기본 항목: 식장, 스드메(스튜디오/드레스/메이크업), 혼수, 예물, 예단, 신혼여행, 한복, 폐백음식, 청첩장, 답례품 등
2. 각 항목은 선택/해제가 가능해야 한다 (필요 없는 항목 제외)
3. 일부 항목은 세부 조건 선택이 가능해야 한다
   - 예: 식장 → 호텔 / 웨딩홀 / 스몰웨딩 / 야외
   - 예: 신혼여행 → 국내 / 동남아 / 유럽
4. 사용자의 지역(서울/경기/지방)을 선택할 수 있어야 한다

### 처리 단계
5. 선택된 항목+조건 조합에 대해 통계 기반 평균 금액을 산출해야 한다
6. 총 예산 합계를 계산해야 한다

### 결과 단계
7. 항목별 예상 금액을 테이블로 보여줘야 한다
8. 항목별 비율을 차트(원형 또는 막대)로 시각화해야 한다
9. 총 예상 예산 합계를 눈에 띄게 표시해야 한다

### 통계 데이터
10. 통계 데이터는 사전에 정의된 정적 데이터로 관리한다 (실시간 시세 연동 제외)
11. 통계 데이터의 출처/기준 시점을 표시해야 한다

## Step 6. Existing vs New Screen
완전히 신규 화면

## Step 7. Access Flow

### 진입점
- 직접 URL 접근
- 공유 링크 유입 (Phase 2)

### 퍼널 흐름
랜딩(/) → CTA 클릭 → 유형 선택(/budget-draft) → 결과 확인

### 이탈점
- 랜딩에서 이탈 (관심 없음)
- 입력 중 이탈 (선택이 복잡하거나 항목을 모르겠음)
- 결과 확인 후 이탈 (목적 달성 — 성공적 이탈)

### URL 구조
- `/` — 랜딩
- `/budget-draft` — 유형 선택 + 결과 확인 (한 페이지)

## Step 8. Screen Inventory

### 화면 1: 랜딩 페이지 (`/`)
- 역할: 서비스 소개 + CTA로 입력 화면 유도
- 정보 위계:
  1. 헤드카피 — 서비스 가치 한 줄
  2. 서브카피 — 동작 방식 간단 설명
  3. CTA 버튼 — "내 결혼 예산 만들기"
  4. (선택) 신뢰 요소 — "통계 기반", "3분 완성" 등

### 화면 2: 예산 생성 페이지 (`/budget-draft`)
**상태 A — 입력 (유형 선택)**
- 정보 위계:
  1. 진행 안내 — "어떤 결혼을 계획하고 계세요?"
  2. 지역 선택 — 서울/경기/지방
  3. 항목 선택 — 토글/체크 방식
  4. 세부 조건 — 하위 옵션
  5. CTA — "예산 초안 만들기"

**상태 B — 결과 확인**
- 정보 위계:
  1. 총 예상 예산 — 큰 숫자로 강조
  2. 항목별 금액 테이블
  3. 비율 차트
  4. 통계 출처/기준
  5. (Phase 2) 공유/저장 버튼

## Step 10. UX Writing
✅ 완료

### 랜딩 페이지
- 헤드카피: "내 결혼, 대략 얼마나 들까?"
- 서브카피: "결혼 유형만 선택하면 통계 기반으로 항목별 예산 초안을 자동으로 만들어드려요. 시세 조사 없이, 3분이면 전체 예산 감을 잡을 수 있어요."
- CTA: "내 결혼 예산 만들기"
- 신뢰 요소: 📊 통계 기반 · ⏱ 3분 완성 · 🔒 로그인 불필요

### 입력 화면
- 타이틀: "어떤 결혼을 계획하고 계세요?"
- 안내: "해당하는 항목을 선택해주세요. 선택한 항목에 맞는 예산 초안을 만들어드릴게요."
- 지역: 서울 / 경기 / 지방
- 항목: 식장, 스드메, 혼수, 예물, 예단, 신혼여행, 한복, 폐백음식, 청첩장, 답례품, 기타
- 세부 조건: 식장(호텔/웨딩홀/스몰웨딩/야외), 신혼여행(국내/동남아/유럽/기타)
- CTA: "예산 초안 만들기"
- 미선택 시: "항목을 1개 이상 선택해주세요" (CTA 비활성)

### 결과 화면
- 총 예산: "총 예상 예산" / "약 {금액}만원"
- 조건 요약: "{지역} · {항목들} 기준"
- 테이블: 항목 / 세부조건 / 예상 금액
- 차트: "항목별 비율"
- 뒤로가기: "← 다시 선택하기"
- 출처: "* 통계 출처: 2025년 결혼비용 실태조사 (듀오) 기준, 실제 금액은 업체/조건에 따라 달라질 수 있습니다."
- 에러: "예산 정보를 불러오지 못했어요. 다시 시도해주세요."
