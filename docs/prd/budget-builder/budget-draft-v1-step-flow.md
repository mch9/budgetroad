# 예산 초안 v1 — STEP 플로우 스펙

## 개요
기존 한 화면 입력 방식을 6단계 STEP 플로우로 변경.
선택지를 하나씩 제공하고, 마지막에 결과를 보여주는 구조.

## STEP 구조

### STEP 1: 지역
- 선택지: 경기도 / 서울(강남) / 서울(강남외)
- 택1

### STEP 2: 예식
- 예식장 유형: 컨벤션 / 채플 / 하우스 / 야외 (택1)
- 예식 시기: 성수기(3~6월, 9~11월) / 비성수기(7~8월, 12~2월) (택1)

### STEP 3: 스드메
- 스튜디오: 심플 / 스탠다드 / 럭셔리 (택1)
- 드레스: 심플 / 스탠다드 / 럭셔리 (택1)
- 메이크업: 심플 / 스탠다드 / 럭셔리 (택1)

### STEP 4: 식사
- 예상 보증인원: 구간 선택 (100명/200명/300명/400명+) + 직접 입력 옵션
- 1인당 식사 비용: 구간 선택 (5만/7만/10만) + 직접 입력 옵션

### STEP 5: 예물/예단
- 예물: 실속형(300만) / 대중형(650만) / 프리미엄형(1000만) / 생략 (택1)
- 예단: 유(평균 금액) / 생략 (택1)

### STEP 6: 신혼여행
- 유 / 무 (택1)
- 유 선택 시: 기본 1000만원, 사용자가 원하면 금액 수정 가능

### 결과
- 항목별 비중 도넛 차트 (총 예산 중앙 표시)
- 항목별 예산 상세 리스트 (아이콘 + 항목명 + 금액 + 가격대 범위 바)
- 생략한 항목은 '미포함'으로 표시
- 하단 CTA: "조건 바꿔서 재계산하기" + "결과 공유하기"
- 면책 문구: "위 예산은 최근 시장 평균 데이터를 기반으로 한 참고용 추정치입니다."

## UX 규칙
- STEP 간 이동: '다음' 버튼 클릭으로 진행
- 뒤로가기: 이전 STEP으로 돌아가기 가능 (선택값 유지)
- 상단 프로그레스 바로 현재 위치 표시
- 각 STEP에 질문 헤더 + 보조 설명 텍스트

## 디자인 참고
- 카드 기반 선택 UI (아이콘 + 제목 + 설명 + 라디오)
- 선택된 카드: 브랜드 컬러(#FF8400) 테두리 + 체크 아이콘
- 미선택 카드: 회색 테두리
- 모바일 퍼스트, 데스크톱 반응형
