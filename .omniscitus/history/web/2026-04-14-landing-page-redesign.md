# 랜딩 페이지 공감형 재구성

## Summary
랜딩 페이지를 공감형 5섹션에서 피그마 기반 풀스크린 인트로로 전면 교체. 라인아트 일러스트 + 대형 헤드라인 + 다크 pill CTA 구성.

## Context
- **Background**: 기존 랜딩이 "예산 초안 만들기"라는 실용적 CTA만 있어서 진지하게 결혼을 준비하는 사람만 유입됨. 결혼 시장의 정보 접근성 문제에 대한 공감 부재
- **Requirements**: 공감형 톤 + 가벼운 CTA; 결혼 비용 정보의 높은 진입장벽에 대한 공감 메시지; SEO 키워드(결혼 비용, 결혼 예산, 웨딩 비용 계산기 등) 자연 포함
- **Decisions**: CTA 문구 "내 결혼 예산 구경하기 →" (구경이라는 단어로 부담 낮춤); 하단에 "아직 계획 없어도 괜찮아요" 문구 추가; 5섹션 구성 (히어로/공감/작동방식/신뢰/하단CTA)
- **Constraints**: 현재 정적 페이지로 구성, 동적 요소 없음
